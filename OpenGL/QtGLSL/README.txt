一、目录说明

common 目录下放置公共代码
libs   目录下放置用到的库

lessonX 该目录下会有一个RootProj管理所有的子项目YYY
build   YYY的build目录用来存放编译产生的文件，其下包含build-Debug build-Release。
res     YYY的res目录用来存放程序读取

二、文件说明
README.txt  每个lessonX下有一个README.txt，记录相关的笔记。
TIPS.txt    根目录下有TIPS.txt，记录一些注意事项。

三、内容说明
该项目记录了《OpenGL Programming Guide,Eighth Edition》读书笔记，实现了书中的程序。项目使用了Qt5.
